# AEMET-DATOS_PLUVIOMETRICOS

Script python2 para recolectar los datos de las estaciones con datos pluviométricos de la Agencia Estatal de Meteorología (AEMET) de Espàña, en su plataforma AEMET Open Data (https://opendata.aemet.es/centrodedescargas/inicio)
Es necesaria una api key que se puede obtener de forma gratuita registrándose en el siguiente enlace: 
https://opendata.aemet.es/centrodedescargas/altaUsuario?

Script python2 to collect the data of the stations with pluviometric data from the Spanish Meteorological Agency (AEMET), in its AEMET Open Data platform (https://opendata.aemet.es/centrodedescargas/inicio)

https://opendata.aemet.es/centrodedescargas/altaUsuario?
